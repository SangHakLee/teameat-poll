## 디자인 시스템

### 테마: 참이슬 (Chamisul Theme)

회식투표 앱은 한국의 대표 소주 브랜드인 "참이슬"에서 영감을 받은 초록색 기반 디자인 시스템을 사용합니다.

### 색상 팔레트

| 용도 | 색상 코드 | HSL | 설명 |
|------|----------|-----|------|
| **Primary** | `#24752c` | `125 52% 30%` | 참이슬 초록색 - 메인 브랜드 컬러 |
| **Secondary** | `#00b3ed` | `195 100% 46%` | 파란색 - 액센트, 탭, 버튼 |
| **Destructive** | `#be1e2b` | `355 76% 43%` | 빨간색 - 삭제, 불가, 경고 |
| **Success** | `#24752c` | `125 52% 30%` | Primary와 동일 - 성공, 가능 |
| **Background** | `#f7f7f6` | `145 12% 97%` | 연한 그레이 배경 |
| **Foreground** | `#1f3d21` | `140 30% 15%` | 진한 초록 텍스트 |
| **Muted** | `#eaebe9` | `145 8% 92%` | 비활성화, 보조 배경 |
| **Border** | `#dfe0de` | `145 12% 88%` | 경계선 색상 |

### 타이포그래피

```css
/* 기본 폰트 설정 */
--font-main: 'GabiaDunn', 'Pretendard', -apple-system, BlinkMacSystemFont, system-ui, sans-serif;

/* 폰트 크기 */
html {
  font-size: 120%; /* 16px → 19.2px (가독성 향상) */
}
```

**폰트 패밀리:**
- **GabiaDunn** (메인): 손글씨 느낌의 부드러운 폰트
- **Pretendard**: 한글 폴백 폰트
- **시스템 폰트**: 최종 폴백

### Border Radius

```css
--radius: 1rem; /* 16px - 부드러운 곡선 */

/* 파생 값 */
lg: var(--radius)           /* 16px */
md: calc(var(--radius) - 2px) /* 14px */
sm: calc(var(--radius) - 4px) /* 12px */
```

### 그림자 & 효과

```css
/* 따뜻한 그림자 */
.shadow-warm {
  box-shadow:
    0 1px 2px hsl(140, 20%, 20%, 0.04),
    0 4px 16px hsl(140, 20%, 20%, 0.06);
}

/* 카드 그림자 */
.shadow-card {
  box-shadow: 0 1px 3px hsl(140, 20%, 20%, 0.05);
}

/* 부유 효과 */
.shadow-float {
  box-shadow:
    0 4px 6px -1px hsl(140, 30%, 20%, 0.08),
    0 2px 4px -2px hsl(140, 30%, 20%, 0.04);
}

/* 그라데이션 배경 */
.gradient-warm {
  background: linear-gradient(180deg,
    hsl(145, 12%, 97%) 0%,
    hsl(145, 15%, 94%) 100%);
}
```

### 애니메이션

```css
/* 페이드 인 */
@keyframes fade-in {
  from { opacity: 0; transform: translateY(8px); }
  to { opacity: 1; transform: translateY(0); }
}

/* 스케일 인 */
@keyframes scale-in {
  from { opacity: 0; transform: scale(0.95); }
  to { opacity: 1; transform: scale(1); }
}

/* 아코디언 */
@keyframes accordion-down {
  from { height: 0; }
  to { height: var(--radix-accordion-content-height); }
}

- 주말 제외: 토요일, 일요일 비활성화

- 빨간색 텍스트
- 하단에 공휴일 이름 (8px 폰트)

        링크를 복사해서 참여자에게 공유하세요
      </DialogDescription>
    </DialogHeader>
    <DialogFooter>
      <Button onClick={handleClose}>닫기</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

**모달 목록:**
